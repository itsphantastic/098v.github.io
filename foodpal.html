FoodPal is a mobile application that helps answer the day old question:
What do you want to eat?

ABOUT
FoodPal seeks to aid both restuarants and customers dining experience. FoodPal helps connect users to their most desired dining experience based on out interllengent algorithm programming. Where the more the user uses the application, the more it adapts to the users what the users likes and dislikes are. Providing the user with more appropriate choices the application can make. It also encouraages new experiences for the user to try. Each individual experience will be different from one another since everyone likes different foods.

REASON
With all the mobile applications to find places to eat, individuals are still scrolling trying to find the right answer, while using traditional methods.

My Role

I was the creator and visionary in creating the mobile application. FoodPal was created to help improve the experience between customer and restuarants. Elimiting the constant scrolling when it comes to trying to pick somewhere to eat. This is an on going project working to improve user experience and the overall design. The FoodPal started in 2016, learning and growing as a designer there is always something to improve on. I am currently redesigning and we working the formula to bring you a better FoodPal.

Challenges

Sperating ourselves between the multitude of mobile applications users can already choose from.

Coming up with a new way to bring the dining and seaching experince. Using an augmented reality for users to use both their movement and the map GPS, to curate appropriate restuarants for each individual users.

Having an advance aligorthm where the AI is able to learn from what interactons each individual users.

Mission Statement

Stop Scrolling and Start Enjoying the finer things in life.

Solution

The solution is by making an mobile application that can learn each individuals users needs and wants. Bring them closer to their more desired restaurant choices.

Design Deliverables
Creating a mobile application, is the first step into bring this platform to our users.

With the current fast pace target market of millennials, research shows that millennials spend approximately 3 hours a day on their phones alone. Using unique augment reality to display the distance between user and restaurants.

Using an appealing UX/ UI to speak volumes to the target market. Simple straight forward FoodPal wants to help you solve the day old queston what do you want to eat!

Lessons Learned

Take the view through the path of least interested. Looking for new ways to communicate the proper information, to both front end-user and back end-user. Knowing your audience really helps keep the design direction and choices on the correct path. Looking from a new outlook to create something both end-users and backend-users would be happy to be apart of. I tried to articulate the clearest path for the deliverables.
